# LilaFit - Proyecto Final React JS

E-commerce desarrollado como proyecto final del curso de React JS en CoderHouse.

## Funcionalidades

- Catálogo de productos
- Filtro por categorías
- Detalle de producto
- Carrito de compras
- Checkout con confirmación de orden
- Navegación SPA con React Router
- Manejo de estado global con Context API

## Tecnologías utilizadas

- React
- React Router
- Context API
- CSS

## Instalación

npm install
npm run dev
